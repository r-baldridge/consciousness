# Philosophical Traditions Index

## Overview

This document indexes the world's philosophical traditions without imposing a simplistic East/West binary. Philosophy has emerged across all human cultures, with rich cross-pollination throughout history. We organize by:

1. **Cultural-Geographic Region** - Where traditions emerged (not a value hierarchy)
2. **Historical Era** - When traditions flourished
3. **Methodological Approach** - How philosophy is conducted
4. **Core Questions** - What problems traditions address

## By Methodological Approach

### Analytic-Logical
Emphasis on clarity, argument, formal methods
- Analytic tradition, Nyaya, parts of Buddhist logic

### Dialectical
Truth through opposition and synthesis
- Hegelian, Madhyamaka, some Daoist approaches

### Phenomenological
Direct description of lived experience
- Husserlian phenomenology, Buddhist abhidharma, contemplative approaches

### Contemplative-Meditative
Knowledge through disciplined practice
- Yoga, Buddhist meditation theory, Sufi, Hesychasm, Daoist cultivation

### Hermeneutic
Interpretation of texts and meaning
- Gadamerian hermeneutics, traditional Vedantic commentary, Talmudic method

### Apophatic/Negative
Knowledge through negation of what cannot be said
- Pseudo-Dionysius, Advaita "neti neti", Madhyamaka, Meister Eckhart

### Dialogical
Truth through encounter and conversation
- Socratic method, Buber's I-Thou, Ubuntu dialogue

### Narrative
Philosophy through story and metaphor
- Indigenous traditions, Zhuangzi, Kierkegaard

## Cross-Traditional Themes

### Questions All Traditions Address

| Question | Example Traditions Engaging |
|----------|---------------------------|
| What is real? | Platonic, Advaita, Madhyamaka, Falsafa |
| How do we know? | Empiricism, Nyaya, Yogacara, Kalam |
| How should we live? | Stoicism, Confucian, Ubuntu, Buddhist ethics |
| What is the self? | Cartesian, Buddhist no-self, Advaita, Ubuntu |
| What is consciousness? | Phenomenology, Yogacara, Samkhya, Process |
| What is liberation? | Stoic, Buddhist, Jain, Hindu soteriology |

### Historical Cross-Pollination

- **Greek-Indian contact**: Pyrrho influenced by Indian thought; Buddhist logic parallels
- **Islamic transmission**: Aristotle preserved and developed through Arabic
- **Buddhist-Daoist synthesis**: Chan/Zen emergence
- **German-Indian**: Schopenhauer, early Indology
- **Kyoto School**: Explicit Buddhist-Western synthesis
- **Contemporary global**: Cross-traditional dialogue increasing

## Implementation Notes

### Data Model Implications

Each tradition should have:
- `primary_region`: Where it emerged
- `secondary_regions`: Where it spread/influenced
- `era`: Historical period
- `methods`: List of methodological approaches
- `core_questions`: Philosophical domains addressed
- `influences`: Traditions it drew from
- `influenced`: Traditions it shaped

### Avoiding False Equivalences

When synthesizing across traditions:
- Acknowledge genuine differences in frameworks
- Note where concepts don't map cleanly
- Respect tradition-internal diversity
- Track fidelity to each source tradition

### Continuous Expansion

This index is necessarily incomplete. Priority additions:
- More African traditions (Yoruba, Zulu, etc.)
- More Indigenous traditions
- More Islamic sub-traditions
- Chinese Buddhism (distinct from Indian and Japanese)
- Korean Neo-Confucianism
- Southeast Asian Buddhist traditions
