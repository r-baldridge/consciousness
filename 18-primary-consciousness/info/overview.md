# Form 18: Primary Consciousness - Overview

## Foundational Theory of Primary Conscious Experience

### Introduction

Primary Consciousness represents the most fundamental level of conscious experience - the basic awareness that transforms mere information processing into felt, subjective experience. Form 18 implements the theoretical and computational foundations of primary consciousness, establishing the critical bridge between unconscious processing and conscious awareness that underlies all higher-order consciousness forms.

Primary consciousness is distinguished from higher-order consciousness (meta-consciousness, narrative consciousness) by its immediate, unreflective nature. It provides the foundational "what-it-is-like-ness" of experience - the qualitative, subjective character that makes mental states conscious rather than merely computational.

### Core Theoretical Framework

#### 1. Defining Primary Consciousness

Primary consciousness encompasses several fundamental characteristics:

**Phenomenal Awareness**: The basic subjective experience that accompanies conscious states - the "feeling" of seeing red, the "sensation" of pain, the "experience" of thinking.

**Qualitative Experience (Qualia)**: The intrinsic qualitative properties of conscious experiences - the redness of red, the painfulness of pain, the specific character of each conscious moment.

**Subjective Perspective**: The first-person perspective from which all conscious experience is undergone - the sense that there is "someone" experiencing.

**Present-Moment Awareness**: The immediate, temporally-located nature of conscious experience - consciousness is always of the present moment.

**Unified Field of Experience**: The integration of diverse sensory and cognitive inputs into a single, coherent conscious experience.

#### 2. Theoretical Foundations

Primary consciousness draws from multiple theoretical traditions:

**Integrated Information Theory (IIT)**: Consciousness corresponds to integrated information (Î¦) - the amount of information generated by a system above and beyond its parts. Primary consciousness emerges when information integration reaches sufficient complexity and coherence.

**Global Workspace Theory (GWT)**: Consciousness arises from global broadcasting of information across multiple specialized processors. Primary consciousness represents the basic workspace where unconscious specialized modules compete for global access.

**Attention Schema Theory**: Consciousness is the brain's model of its own attention processes. Primary consciousness represents the fundamental attention schema that creates subjective awareness.

**Predictive Processing**: Consciousness emerges from predictive models of sensory input and their prediction errors. Primary consciousness is the basic predictive model of immediate experience.

**Higher-Order Thought Theory**: Consciousness requires higher-order representations of mental states. Primary consciousness represents the basic meta-representational capacity.

#### 3. Neural Correlates and Implementation Principles

Primary consciousness manifests through specific neural mechanisms:

**Thalamocortical Integration**: The thalamus serves as a central integration hub, binding diverse cortical areas into unified conscious experience.

**Gamma-Band Synchronization**: 40Hz gamma oscillations facilitate the temporal binding necessary for unified conscious experience.

**Global Ignition Patterns**: Widespread cortical activation patterns that correspond to conscious access and global broadcasting.

**Default Mode Network**: The brain's default processing mode that maintains basic conscious awareness and self-reference.

**Attention Networks**: Executive, alerting, and orienting attention systems that structure and direct conscious experience.

    async def _compute_self_reference_strength(self,
                                             phenomenal_content: PhenomenalContent,
                                             context: Dict[str, Any]) -> float:
        """Compute strength of self-reference in conscious experience."""

        # Analyze phenomenal content for self-referential elements
        self_ref_elements = []

        # Body-related self-reference
        if 'proprioceptive' in phenomenal_content.qualia_descriptors:
            self_ref_elements.append(
                phenomenal_content.qualia_descriptors['proprioceptive']['body_awareness']
            )

        # Visual self-reference (e.g., seeing own body parts)
        if 'visual_self_elements' in phenomenal_content.qualia_descriptors:
            self_ref_elements.append(
                phenomenal_content.qualia_descriptors['visual_self_elements']['self_visibility']
            )

        # Cognitive self-reference
        if 'self_related_thoughts' in context:
            self_ref_elements.append(context['self_related_thoughts']['self_relevance'])

        # Compute aggregate self-reference strength
        if self_ref_elements:
            self_reference_strength = np.mean(self_ref_elements)
        else:
            self_reference_strength = 0.3  # Default minimal self-reference

        return min(1.0, max(0.0, self_reference_strength))
